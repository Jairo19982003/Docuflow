# DocuFlow Systems - Prototipo Web Empresarial

## 📋 Descripción del Proyecto

DocuFlow Systems es una empresa ficticia de gestión documental y digitalización. Este prototipo web presenta:

1. **Equipo de Trabajo**: 5 integrantes con puestos definidos de una empresa de software
2. **Productos de Software**: Sistema de gestión documental (DocuFlow DMS)
3. **Arquitectura de Software**: Diagrama visual con iconos del flujo de procesos
4. **Servicios Tecnológicos**: Servicios de digitalización de documentos
5. **Ubicación Física**: Quito, Ecuador - Av. Amazonas N24-03
6. **Diagrama Empresarial**: Organigrama completo con descripción de puestos

## 📦 Instalación y Ejecución

### Prerequisitos
- Node.js (versión 18 o superior)
- npm (viene con Node.js)

### Pasos para Ejecutar

1. **Navegar al directorio del proyecto**
```powershell
cd "docuflow-systems"
```

2. **Instalar dependencias** (si es necesario)
```powershell
npm install
```

3. **Ejecutar el servidor de desarrollo**
```powershell
npm run dev
```

4. **Abrir en el navegador**
- La aplicación estará disponible en: `http://localhost:5173`

### Comandos Disponibles

```powershell
# Desarrollo
npm run dev          # Inicia servidor de desarrollo

# Producción
npm run build        # Construye para producción
npm run preview      # Previsualiza build de producción

# Linter
npm run lint         # Ejecuta ESLint
```
